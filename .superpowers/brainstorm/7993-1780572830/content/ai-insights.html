How the AI layer works

  • A Supabase Edge Function aggregates statistics server-side then calls Claude API — raw logs never leave your database
  • Claude receives only anonymised summaries (averages, counts, durations) — not individual entries
  • Refreshes once daily or on-demand — keeps API cost minimal (~$0.01–0.05/month)
  • Home nudge shows the single most relevant insight · Full screen shows all categories
  • Three severity levels: Normal · Watch this · Contact PD
